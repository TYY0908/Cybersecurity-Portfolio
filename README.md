# Cybersecurity Portfolio

## About Me
Cybersecurity and IT student building practical, beginner-friendly security tools with Python. This portfolio focuses on hands-on learning in defensive security, basic networking, password security, and log analysis.

## Projects

| Project | Focus Area | What It Demonstrates |
| --- | --- | --- |
| [Simple Vulnerability Scanner](vulenrability_scanner) | Network Security | Scans common TCP ports and reports open services for authorized targets. |
| [Hash Cracker](hash_cracker) | Password Security | Demonstrates dictionary-based MD5 hash matching in a lab environment. |
| [Password Strength Checker](password_strength_checker) | Secure Authentication | Scores password strength using length, character variety, and common-pattern checks. |
| [Security Log Analyzer](security_log_analyzer) | SOC Fundamentals | Parses sample authentication logs and identifies failed login patterns. |
| [Phishing URL Checker](phishing_url_checker) | Web Security Awareness | Flags suspicious URL traits commonly seen in phishing attempts. |

## Skills Practiced

- Python scripting
- Socket programming
- File parsing and log analysis
- Password security fundamentals
- Basic threat detection logic
- Git and GitHub project documentation

## Learning Goal

Each project is intentionally small and explainable. The goal is to show consistent hands-on practice and the ability to build, document, and improve cybersecurity tools over time.
