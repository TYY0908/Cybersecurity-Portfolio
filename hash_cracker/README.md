# Hash Cracker

## Overview
A beginner cybersecurity project that demonstrates dictionary-based hash matching using Python and a custom wordlist.

## Features
- MD5 hash comparison
- Dictionary attack simulation in a controlled lab
- Custom wordlist support
- Clear result message when a match is found

## Technologies Used
- Python
- hashlib
- pathlib

## How to Run

```bash
python hash_cracker.py
```

## Ethical Use

This project is for password security education and authorized lab testing only. It should not be used against accounts, systems, or data you do not own or have permission to test.
