# Password Strength Checker

## Overview

A beginner-friendly Python tool that evaluates password strength and gives simple improvement suggestions.

## Features

- Checks password length
- Detects uppercase, lowercase, numbers, and symbols
- Flags common weak password patterns
- Provides a readable strength score

## Technologies Used

- Python
- re
- getpass

## How to Run

```bash
python password_strength_checker.py
```

## Learning Focus

This project demonstrates secure authentication basics and shows how password rules can be translated into simple detection logic.
