# Simple Vulnerability Scanner

## Overview
A beginner cybersecurity project built with Python that scans common TCP ports on an authorized target system.

## Features
- Port scanning
- Open/closed port detection
- Socket programming
- Beginner vulnerability assessment
- Timeout handling for slow connections

## Technologies Used
- Python
- Socket library

## How to Run

```bash
python scanner.py
```

## Ethical Use

Only scan systems you own or have explicit permission to test. Unauthorized scanning can violate laws, policies, or terms of service.
